#  Trainings
Репозиторий содержит прмеры кода на современном С++ и реализацию некоторых нетривальных алгоритмов на С++.
Для сборки необходимо:
- cmake.
- компилятор с поддержкой С++20. 
- фреймворк Catch2

Для установки Catch2 можно воспользоваться менеджером пакетов `vcpkg`. Пример скрипта в `scripts\installPackages.bat`
Для того чтобы подтянуть библиотеки установленные с помощью `vcpkg`, пропишите в переменную среду `VCPKG_ROOT` путь к директории `vcpkg`


## Algorithms

### AStar
Реализация алгоритма А*. А* - алгоритм поиска пути в графе. Данный алгоритм как правило быстрее классического алогоритма Дийкстры, но требует информации о координатах вершин (для вычисления расстояния (евклидова или манхеттенского)). 

### Contraction
Рализация Contraction Hierarchy Algorithm - по сути двунаправленный алгоритм Дийкстры с препроцессингом графа. Требует препроцессинга графа (применения ряда эвристик), но затем работает ощутимо быстрее.

### DisJointSet
Рализация структуры данных неперескающиеся множества и алгоритма Крускаля для построения остовного дерева графа.

## CppFeatures
Особенности языка С++

### OperatorOverload
Пергрузка операторов :  Three way comparisson  (он же spaceShipOperator <=>), оператора double, оператора литерала, и немного специализации шаблонов.
### Range
Наивная реализция range.
### Vectors
Рализация динамического массива на С++, он же vector. Вектор демонстрирует множество возможностей С++ и способов выстрелить себе в ногу: работа с памятью, безопасность исключений, реализация специальных функций членов (конструкторы, деструктор, операторы присваивания), move/copy семантика.   
### Preprocessor
Немного особенностей препроцессора С++
